Overview
========

wnutils is a python package for reading and plotting
`webnucleo <http://webnucleo.org/>`_ files.

Installation
------------

Install from `PyPI <https://pypi.org/project/wnutils>`_ with pip by
typing in your favorite terminal::

    $ pip install wnutils
